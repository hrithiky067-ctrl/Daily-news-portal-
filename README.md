# Daily News Portal

## Features
- Latest News Updates
- User-friendly Interface
- Categories: Sports, Technology, Entertainment, etc.
- Search Functionality

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/hrithiky067-ctrl/Daily-news-portal- 
   ```  
2. Navigate to the project directory:
   ```bash
   cd Daily-news-portal-
   ```  
3. Install the required dependencies:
   ```bash
   npm install
   ```  
4. Start the development server:
   ```bash
   npm start
   ```

## Usage Guide
1. Open your browser and go to `http://localhost:3000`.
2. Browse the latest news and select categories for more specific articles.
3. Use the search bar to find specific news articles by keywords.
